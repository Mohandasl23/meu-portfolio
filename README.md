# 🚀 Meu Portfólio Profissional - Dev & Automação

Este é o repositório do meu portfólio pessoal, focado em apresentar soluções tecnológicas que transformam processos operacionais complexos e manuais em fluxos ágeis, integrados e 100% automatizados.

🎯 **Acesse o site ao vivo:** [https://mohandasl23.github.io/meu-portfolio/](https://mohandasl23.github.io/meu-portfolio/)

---

## 💼 Sobre Mim & Diferencial de Mercado
Com uma sólida trajetória na **área contábil**, desenvolvi uma visão analítica apurada para mapear gargalos operacionais e fluxos de negócios. Hoje, utilizo essa bagagem de mercado para codificar soluções reais utilizando o ecossistema **Python**, com foco absoluto em eficiência, eliminação de erros humanos e ROI.

## 📈 Projetos em Destaque Apresentados

1.  **NEW_ACCOUNTING_ENTRIES (Python / Playwright / PDFPlumber / OpenPyXL)**
    *   Leitura automatizada de extratos em PDF, filtragem algorítmica e injeção de lançamentos contábeis direto via ERP.
2.  **Automação de Relatórios Gerenciais (Python / Playwright)**
    *   Bot utilitário para autenticação segura e extração massiva de relatórios em sistemas legados.
3.  **API REST & Hubs Full Stack (Node.js / Express / MongoDB / React Native)**
    *   Ecossistema back-end para gerenciamento de requisições e controle de acessos de aplicações mobile.
